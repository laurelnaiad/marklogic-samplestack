allTagsDialog

  • - dialog in module app

Displays all tags in the database in a paged modal, and allows for searching and selecting the tags for inclusion as filters on a search. Uses ui.bootstrap.modal. Also see the allTagsDialogCtlr and the allTagsStartFrom filter.

Usage

allTagsDialog(unselTags, selTags);

Arguments

Param Type Details
unselTags Array.<object>

unselected tags
selTags Array.<object>

selected tags

Returns

angular.Promise

the promise will either be rejected or will resolve to an object the following properties that reflect the result of the user interaction has the dialog:

  • unselTags - {Array.<object} the unselected tags
  • selTags - {Array.<object} - the selected tags
