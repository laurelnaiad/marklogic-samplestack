allTagsStartFrom

  • - filter in module app

Returns the starting element for columnar display of tags. See allTagsDialog.

Usage

In HTML Template Binding

{{ allTagsStartFrom_expression | allTagsStartFrom : startIndex : colOffset}}

In JavaScript

$filter('allTagsStartFrom')(tags, startIndex, colOffset)

Arguments

Param Type Details
tags Array.<object>

set of tags
startIndex integer

starting index based on current page
colOffset integer

offset based on the column index

Returns

object

The starting tag for the column
