contributorDialogCtlr

  • - controller in module app

Controller for the contributorDialog. The controller is injected by the $modal service.

Upon instantiation the contributorDialogCtlr looks up the specified contributor by id on the server.

Usage

contributorDialogCtlr($scope, $modalInstance, ssContributor, contributorId);
