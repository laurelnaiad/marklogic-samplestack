loginDialogCtlr

  • - controller in module app

Controller for loginDialog. The controller is injected by the $modal service.

Used to provide a user interface through which to autenticate a user.

Upon instantiation the loginDialogCtlr creates an empty instance of ssSession for handling authentication.

Usage

loginDialogCtlr($scope, $modalInstance, ssSession, mlAuth);

Arguments

Param Type Details
$scope angular.Scope

(injected)
$modalInstance ui.bootstrap.modal.$modalInstance

(injected)
ssSession object

Session object
mlAuth object

Authentication object

Methods

  • method:$scope.authenticate();

    Initiate authentication of the $scope.session credentials.

    On success, closes the dialog.

    On failure, reports the failure to the user via the $scope.error property.

  • method:$scope.cancel();

    Dismisses the dialog.

Properties

  • $scope.error

    string

    If present, indicates textually what error occured while attempting to authenticate a user.
