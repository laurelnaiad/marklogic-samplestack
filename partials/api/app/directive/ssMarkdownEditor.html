ssMarkdownEditor

  • - directive in module app

Enables editing of Markdown content with preview tab and editing controls in a button-bar. Uses ssMarkdown for preview and and ngMarkdown for button bar functionality.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <ANY
  ss-markdown-editor="">
...
</ANY>

Arguments

Param Type Details
ssMarkdownEditor expression

the content to which to bind.
