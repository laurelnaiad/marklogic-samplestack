ssAccountDropdown

  • - directive in module app

Display info for the logged-in user, and allows the user to logout of their session.

The Bootstrap Dropdown directive is used to display the dropdown.

The template for this dropdown reads logged-in user information from the scope.store.session object, an instance of ssSession.

scope properties/methods:

Variable Type Details
logout function Emits a logout event
store.session ssSession As provided by $rootScope, provides session information.

Directive Info

  • This directive executes at priority level 0.
