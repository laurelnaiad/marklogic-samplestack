Service components in _marklogic

Name Description
mlHttpInterceptor

Handles CSRF tokens for all AJAX requests. May be configured or disabled in the mlHttpInterceptorProvider.
mlModelBase

Service from which domain-specific model elements are derived.
mlSchema

Manages JSON schemas and enables validation of instances against the schema.
mlStore

Simple object store which attaches itself to the $rootScope so that long-lived model data may be stored on a "side" scope.
mlWaiter

Establishes promise-based properties under the non-enumerable $ml property of model objects such that template code may track the status of promises (usually related to $http requests) that affect the model element instance.
mlAuth

Facilities for handling authentication and sessions.
mlUtil

Provides utility objects and methods.
