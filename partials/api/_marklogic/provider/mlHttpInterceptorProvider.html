mlHttpInterceptorProvider

  • - provider in module _marklogic

Configures CSRF handling in the mlHttpInterceptor.

Properties

  • disableCsrf

    boolean

    If true, CSRF handling on requests will be suppressed and CSRF tokens on responses will be ignored. Defaults to false (CSRF enabled).

  • endpoint

    string

    Configured the endpoint for requesting the CSRF token. Defaults to '/v1/sesion'.

  • headerName

    string

    The header name for the CSRF token. Defaults to X-CSRF-TOKEN.
